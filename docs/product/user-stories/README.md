# RouteScout User Stories

This directory breaks the initial product scenarios into separate files so they can evolve
independently.

The stories now use the shared RouteScout terminology:
- `category` = a grouping/filtering property of a plan
- `plan` = the user's reusable planning request
- `route` = the generated running output
