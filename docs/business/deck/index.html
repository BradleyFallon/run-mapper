Opportunity

Why this product should exist

  • Generic map apps are not built for running route quality.
  • Run trackers help after the run, not before it.
  • Traveling runners and serious runners both have real route-selection pain.
  • The product should create the right route for you, not just show one somebody else uploaded.
Who it serves first Travelers, explorers, and technical runners
Core promise Get the run you would have planned for yourself if you had the time
Position Custom route scout, not generic mapping utility or route database

Product concept

How RouteScout works

1

Choose a Scout Spec

Users start from a planning spec like Safe Early Morning or Race Prep Hills.

2

Scout the best option

RouteScout evaluates multiple candidates against distance, terrain, surface, simplicity, route character, and the runner's actual priorities.

3

Save what is worth keeping

Users compare Route Cards, save favorites, and build a collection of routes that were originally created for them.

Business model

Free gets habit. Plus gets depth. Pro gets intelligence.

  • Free should be real enough for people to run with RouteScout, track runs, and get attached to it.
  • Plus should unlock planning depth, saved value, and routine usefulness.
  • Pro should exist only when RouteScout can genuinely learn the runner and coach intelligently.
Free Run with RouteScout

Basic tracking, basic route-follow, goal setting, one active Scout Spec, and limited scouting.

Plus Plan with RouteScout seriously

More scouting, collections, multiple Scout Specs, route comparison, route explanations, and trip planning.

Pro RouteScout knows you

Personalized route intelligence, learned preferences, adaptive suggestions, and coaching-style logic.

1 Try it

Scout a route and use it on a real run for free.

2 Get attached

Track runs, set goals, and start wanting more planning power.

3 Upgrade for depth

Subscribe to Plus when planning, saving, and reusing routes becomes routine.

4 Grow into Pro

Move up later when RouteScout can learn preferences and make intelligent recommendations.

Pricing

Cheap to try, affordable to keep

  • We want low-friction adoption and enough value to support hosting, APIs, and launch costs.
  • Free gets people engaged. Plus is the main paid tier. Pro is the later premium intelligence layer.
  • The goal is to keep the product affordable while charging for depth, not for basic use.
Free Use RouteScout enough to feel it

Real run use, capped scouting, one active Scout Spec, and limited saved planning depth.

Plus: $2.99 / month Mainstream paid tier

More scouting, collections, comparisons, explanations, and multiple Scout Specs.

Pro: $4.99 / month Personal intelligence tier

Adaptive recommendations, learned preferences, and coaching-style route suggestions.

Free drives habit and reviews Plus monetizes planning depth, Pro monetizes personalization
