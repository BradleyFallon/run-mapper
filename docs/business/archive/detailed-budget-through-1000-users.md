# RouteScout Detailed Budget Through Release And 1,000 Activated Users

## Purpose

This document breaks the RouteScout funding plan into concrete budget lines from now through:
- shipped web planner
- iPhone MVP release
- early launch
- `1,000` activated users

This is meant to be easier to discuss than a single lump-sum number.

## Budget Summary

Recommended working budget:
- `$42,000`

This is the detailed line-item version of the funding ask.

## Assumptions

This budget assumes:
- a `12-week` concentrated build phase
- `6 months` of operating runway from now through early traction
- founder-led execution across product, design, engineering, brand, and marketing
- a soft launch first, not a heavy paid-acquisition strategy
- milestone bonuses are part of the budget from the beginning

This budget does **not** assume:
- immediate profitability
- high paid conversion in the first weeks
- enterprise-scale hosting

## 1. Developer Pay

- `$12,000`

Assumption:
- `$1,000 / week`
- `12 weeks`

This is the guaranteed developer pay for the execution-heavy development role.

## 2. Developer Bonus Reserve

- `$15,000`

Assumption:
- `$5,000` at `100 activated users`
- `$10,000` at `1,000 activated users`

This should be reserved from the start so it is not treated as an afterthought later.

## 3. Domain, Developer, And Admin Costs

- `$500`

This line should cover:
- domain registration
- Apple Developer Program fee
- minor admin/software/account costs

This is intentionally conservative but not large.

## 4. Hosting, Monitoring, And Infrastructure Reserve

- `$1,200`

Planning assumption:
- average of about `$200 / month` over `6 months`

This should cover:
- app hosting
- database / backend hosting
- monitoring and uptime tools
- basic deployment infrastructure

The cost may start lower and grow as usage increases.

## 5. Routing And LLM API Reserve

- `$1,200`

Planning assumption:
- starts around `$50 / month`
- increases as active usage grows
- averaged across the first `6 months`

This should cover:
- routing service usage
- geocoding / related calls if needed
- LLM usage for route interpretation or planning assistance

This line is intentionally a reserve because actual usage will depend on product behavior and user
volume.

## 6. Development Tools And Creative Software

- `$2,000`

This should cover roughly `6 months` of:
- ChatGPT Pro or comparable high-end AI development support
- Adobe Creative Cloud
- premium software subscriptions needed for development and creative work

This line is deliberately set as a working budget instead of assuming every tool choice is fixed
today.

## 7. Advertising And Promotion Budget

- `$2,300`

Suggested staged plan:
- month 1: `$100`
- month 2: `$200`
- month 3: `$500`
- months 4-6 reserve: `$1,500`

Use this for:
- promoted social posts
- small tests on Google Ads / search capture
- limited paid distribution of creatives that already perform organically

This is a light testing budget, not a large campaign.

## 8. Creative Production And Story Ads

- `$2,000`

This should cover:
- paying runners or amateur actors for simple ad/story videos
- light production support
- editing assets if needed
- stock and voiceover alternatives when useful

This budget should be used selectively.

The strongest content will probably feel like:
- a runner telling a believable story
- not an over-produced commercial

## 9. Freelance Support

- `$2,500`

This should cover specialized help such as:
- SEO / ASO cleanup
- marketing landing-page optimization
- analytics setup help
- occasional technical or design overflow support

This is useful because some tasks are cheaper to delegate than to do from scratch personally.

## 10. Swag And Grassroots Promotion

- `$1,000`

This should cover:
- stickers
- QR-code handouts
- small event materials
- lightweight branded giveaway items

This is not required for launch, but it is useful for community visibility later.

## 11. Contingency Buffer

- `$2,300`

This should cover:
- higher-than-expected API costs
- unplanned tooling expenses
- launch issues
- extra marketing tests
- other small overruns

## Total

- `$42,000`

## Full Ask Vs Initial Deposit

### Full Funding Commitment Ask

- `$42,000`

This is the real ask.

If the company is being financed properly, this is the amount that should be committed to the plan.

### Initial Deposit

- `$15,000`

This is the minimum amount that should be funded into the LLC immediately to begin the build phase
cleanly.

The important distinction:
- `$42,000` is the full funding commitment ask
- `$15,000` is only the first deposit

## Recommended Founder Framing

Use language like:

"I do not want us to underfund this and end up in an unstable half-launched state. I think the
right way to look at it is a detailed budget, not just a vague number. The full ask is $42,000,
because that is what the actual plan requires from here to launch and 1,000 activated users. I
want $15,000 deposited immediately so the build phase can start cleanly, but I do not want us
pretending that is the whole funding need."
